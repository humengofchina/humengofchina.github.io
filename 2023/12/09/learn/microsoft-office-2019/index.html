Microsoft office 2019 批量激活版

一、写在前面：

今天参加学校的小比赛，没想到居然是MSoffice计算机二级的题，在考场不会做题的的感觉。那叫一个难受！

因此，回来后开始下载office全家桶学习，借此机会，出一期Microsoft office2019专业增强版的下载以及激活的步骤。（要不然时间长了我也忘了）

二、下载步骤：

1、首先下载部署工具（浏览器打开，记住下载的位置）

https://www.microsoft.com/en-us/download/details.aspx?id=49117

2、在桌面上新建一个文件夹，命名为office，回车确认。

3、双击打开刚才下载的软件（officedeploymenttool_16731-20398.exe）

同意条款协议（英文的）接着继续continue选择刚才创建的office文件夹，确定。会弹出一个成功的提示，确定即可。

4、接着检查office里面是不是有五个文件（不是五个，关闭杀毒软件后重试上述步骤）

5、接着是最重要的部分——office 版本自定义工具

打开网址：https://config.office.com/deploymentsettings

进行设置的就上述三个大块：（其他都不需要动）

产品和版本：

体系结构：    64位  产品：     office专业增强版2019-批量许可证      Visio专业版 2019 -批量许可证 project 专业版 2019 -批量许可证    语言包：更新频道：    不用管，默认应用：    选择 Word  Excel  PowerPoint 即可，其他没必要

语言：

简体中文（中国）

更新和升级：

把  自动检查更新  关闭即可

完成上述步骤之后点击【导出】

tips：导出以后就会获得一个名字为config.xml的文件（如果你没有显示.xml，那是你没有开启文件扩展名，当然这个没有影响）然后将这个文件移动到一开始创建的office文件夹里面。并复制这个文件夹的路径。

6、快捷键win s搜索cmd，接着以管理员身份运行cmd

输入

cd 刚才复制的office文件路径

然后回车确认，并输入下载命令：

setup /download config.xml

此时光标消失，说明正在运行下载中

tips：（时间会有点长，耐心等待。请不要关闭窗口）（您可以通过任务管理器【快捷键ctrl shift esc】的性能一栏查看实时网速的情况，来检查是否正在下载中）

等到光标闪烁时，说明已经下载好，输入安装命令：

setup /configure config.xml

这个过程就很快了。至此已经下载完成。

三、激活步骤：

**同样快捷键win s搜索cmd，接着以管理员身份运行cmd
**

输入：

cd C:\Program Files\Microsoft Office\Office16

回车确认

cscript ospp.vbs /sethst:kms.v0v.bid && cscript ospp.vbs /act

重启电脑，完成激活！！！

然后把在桌面创建的office文件夹删除即可。
